🔐 Google OAuth — inicio de sesión en el panel

Cómo configurar Google OAuth (paso a paso):
1. Ve a console.cloud.google.com → Credenciales
2. Crea un ID de cliente OAuth 2.0 → tipo Aplicación web
3. URI de redirección autorizado: https://digital.raqmiy.com/auth/callback
4. Pega el Client ID en el campo de abajo y guarda
5. El Client Secret ponlo en Cloudflare Pages → Configuración → Variables de entorno → secreto encriptado con el nombre GOOGLE_CLIENT_SECRET
6. Añade también: SITE_URL = https://digital.raqmiy.com
⚠️ GOOGLE_CLIENT_SECRET no se guarda aquí por seguridad — ponlo como secreto encriptado en el dashboard de Cloudflare Pages.

🖼️ Subida de imágenes (Cloudinary — gratis)

Cómo configurar Cloudinary (gratis, sin tarjeta):
1. Crea cuenta en cloudinary.com
2. En el dashboard copia tu Cloud name (ej: abc123)
3. Ve a Settings → Upload → Upload presets → Add upload preset
4. Signing mode: Unsigned → guarda y copia el nombre del preset
5. Pega ambos valores aquí y guarda
Introduce la contraseña y pulsa Entrar.
